🎓 School Administration System

📌 Project Description

This is a School Administration System built using **Python, Streamlit, and MySQL**.
It helps manage student and teacher data with authentication, department selection, and database operations.

The system provides separate modules for students and teachers to perform different actions.

---

🚀 Features

👨‍🎓 Student Module
* Student Signup & Login
* Store student details in MySQL database
* Select Department & Section
* View Student Details
* Update (Change) Department

👩‍🏫 Teacher Module
* Teacher Signup & Login
* View Teacher Details
* View All Student Details

🏠 Home Page
* School introduction
* Highlights and achievements display

---

🛠️ Technologies Used

* Python
* Streamlit
* MySQL
* Pandas
* Streamlit Option Menu

---

🗄️ Database Used

* MySQL Database (`school_administration`)
* Tables:

  * `student`
  * `teachers`
  * `department`
  * `student_details`

---

▶️ How to Run the Project

1. Clone the repository:
`git clone https://github.com/your-username/your-repo-name.git`

2. Install required libraries:
`pip install streamlit mysql-connector-python pandas streamlit-option-menu tabulate`

3. Setup MySQL database:
* Create database: `school_administration`
* Create required tables

4. Run the application:
`streamlit run app.py`
